# Описания алгоритмов FindRoutes и calculateArrivalTimes

## Алгоритм FindRoutes

Данный алгоритм отвечает за выборку информации о маршрутах общественного транспорта, которые проходят через две указанные остановки в правильном направлении движения. Он обеспечивает формирование списка маршрутов, которые соответствуют заданным условиям и позволяют проехать от начальной остановки до конечной без пересадок. Алгоритм осуществляется за счёт последовательной обработки данных в несколько этапов: 

Получение полного списка всех маршрутов из транспортной системы.

Итерация по всем элементам списка маршрутов.

Проверка наличия начальной остановки в текущем маршруте. – Проверка наличия конечной остановки в текущем маршруте. – Проверка корректности направления движения (начальная остановка должна находиться раньше конечной в последовательности остановок маршрута).

Добавление подходящих маршрутов в результирующий список.

Возврат итогового списка для дальнейшего использования при планировании поездок и построении маршрутов.

## Алгоритм calculateArrivalTimes

Данный алгоритм отвечает за расчёт времени прибытия транспортного средства на каждую остановку маршрута в рамках указанного рейса. Он обеспечивает формирование расписания прибытия на основе заданной средней скорости движения и фиксированных параметров движения между остановками. Алгоритм осуществляется за счёт последовательной обработки данных в несколько этапов: 

Проверка валидности входных параметров (средняя скорость должна быть положительной).

Поиск рейса с указанным идентификатором в системе транспортных рейсов.

Проверка наличия остановок в маршруте выбранного рейса.

Установка времени прибытия на первую остановку равным времени отправления рейса.

Итерация по всем последующим остановкам маршрута.

Расчёт времени в пути между остановками на основе расстояния и средней скорости движения. – Установка времени прибытия на текущую остановку с учётом времени в пути. – Учёт времени стоянки на остановке для расчёта времени отправления к следующей остановке.

Обновление времени прибытия для всех остановок рейса в системе.

Возврат управления после завершения расчётов для дальнейшего использования расписания при планировании поездок.
