# Структура классов алгоритмов в коде

## Расположение классов алгоритмов

### 1. Базовый класс `Algorithm` 
**Файл:** `algorithm.h` / `algorithm.cpp`

Базовый класс для всех алгоритмов, реализует принцип полиморфизма ООП.

```cpp
class Algorithm {
    // Виртуальные методы для полиморфизма
    virtual void execute() = 0;
    virtual std::string getDescription() const = 0;
}
```

### 2. Иерархия алгоритмов поиска пути

**Файл:** `algorithm.h` / `algorithm.cpp`

#### `PathFindingAlgorithm` (уровень 2)
Базовый класс для алгоритмов поиска пути.

#### `BFSAlgorithm` (уровень 3)
- **Алгоритм:** Поиск в ширину (BFS)
- **Назначение:** Поиск всех маршрутов с пересадками
- **Метод:** `findPath()`

#### `FastestPathAlgorithm` (уровень 3)
- **Алгоритм:** Поиск самого быстрого маршрута
- **Назначение:** Выбор оптимального маршрута по времени
- **Метод:** `findPath()`

#### `MinimalTransfersAlgorithm` (уровень 3)
- **Алгоритм:** Поиск маршрута с минимальными пересадками
- **Назначение:** Выбор маршрута с наименьшим количеством пересадок
- **Метод:** `findPath()`

### 3. Другие алгоритмы

#### `ArrivalTimeCalculationAlgorithm`
- **Алгоритм:** Расчет времени прибытия
- **Назначение:** Вычисление времени прибытия на каждую остановку маршрута
- **Метод:** `calculateArrivalTimes()`

#### `RouteSearchAlgorithm`
- **Алгоритм:** Поиск маршрутов между остановками
- **Назначение:** Поиск прямых маршрутов между двумя остановками
- **Метод:** `findRoutes()`

## Использование алгоритмов

### В `JourneyPlanner`
- Использует `BFSAlgorithm`, `FastestPathAlgorithm`, `MinimalTransfersAlgorithm`
- Реализует паттерн Strategy для выбора алгоритма поиска

### В `TransportSystem`
- Использует `ArrivalTimeCalculationAlgorithm` для расчета времени
- Использует `RouteSearchAlgorithm` для поиска маршрутов

## Принципы ООП, реализованные в алгоритмах

1. **Инкапсуляция:** Алгоритмы инкапсулируют логику поиска
2. **Наследование:** Иерархия Algorithm → PathFindingAlgorithm → конкретные алгоритмы
3. **Полиморфизм:** Виртуальные методы позволяют использовать разные алгоритмы через базовый класс
4. **Абстракция:** Базовый класс Algorithm определяет интерфейс для всех алгоритмов
